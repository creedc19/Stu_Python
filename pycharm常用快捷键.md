## PyCharm  常用快捷键：

1. 注释：Ctrl + /
2. 文件重命名：Shift + F6
3. 运行程序：shift + F10
4. 代码格式化：Ctrl + Alt + L
5. 拷贝文件路径：Ctrl + shift + C
6. 调试程序：shift + F9
7. 添加补字符：Alt + 鼠标左键
8. 光标在行开头：Ctrl + [
9. 光标在行结尾：Ctrl + ]
10. 选中当前行：Ctrl + shift + [
11. 取消选中：Ctrl + shift + ]
12. 选择剪贴板内容插入：Ctrl +  Shift + insert
13. 撤销：Ctrl + Z
14. 恢复：Ctrl + Shift + z
15. 复制当前行：Ctrl + D
16. 删除当前行：Ctrl + Y
17. 代码补全：Ctrl + Alt  + 空格
18. 单步执行：F8
